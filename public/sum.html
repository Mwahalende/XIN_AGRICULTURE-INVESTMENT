XIN Agricultural Website - Project Summary

Overview

This website is a full-stack agricultural business management platform for XIN Agricultural. It provides secure admin and customer access, product management, order tracking, and business note-taking, all with a modern UI and robust backend security.

Key Features

  • Admin Authentication: Secure login/register for admins. Only admins can access sensitive pages like the user list (see.html) and admin dashboard.
  • Customer Registration & Login: Customers can register, log in, and manage their orders.
  • Product Management: Admins can upload new products (with images), and all users can view available products and prices.
  • Order Management: Customers can place orders and track their order status. Admins can view and update all orders.
  • Business Notes: Admins can create, view, and manage business notes for internal use.
  • Location Features: Google Maps integration for business location and delivery tracking.
  • Security: Session-based authentication, token-protected admin pages, and forbidden access handling with a custom error page.

How It Works

  • Landing Page: Users arrive at the home page and can choose to log in/register as a customer or admin.
  • Admin Flow: After logging in, admins access the dashboard to manage products, view orders, and handle business notes. Access to sensitive pages (like see.html) is protected by session tokens and only available to logged-in admins.
  • Customer Flow: Customers can register, log in, view products, place orders, and check their order status. They receive email notifications for registration and order updates.
  • Security & Error Handling: Unauthorized access to admin-only pages redirects users to a custom error page (error.html) with a clear message.
  • Modern UI: All pages use a clean, responsive design for a professional user experience.

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MongoDB (with Mongoose)
  • Other: express-session, multer (file uploads), cloudinary (image hosting), Google Maps API, nodemailer (emails)

Summary

This project delivers a secure, feature-rich, and user-friendly platform for XIN Agricultural, supporting both business operations and customer engagement. All sensitive actions are protected, and the site is designed for easy use by both admins and customers.
